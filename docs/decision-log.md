# Clark Decision Log

This log records decisions that have been made strongly enough to guide the project.

## How To Use

- Add one entry per meaningful decision.
- State whether the decision is evidence-backed, assumption-led, or provisional.
- Reference the source docs affected by the decision.
- Update or supersede old entries rather than silently drifting.

## Status Labels

- `Provisional`: current working decision, but still evidence-sensitive
- `Confirmed`: strong enough to treat as default
- `Superseded`: replaced by a later decision
