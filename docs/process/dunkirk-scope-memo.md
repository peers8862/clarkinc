# Dunkirk Scope Memo

## Decision

Dunkirk should be treated as a:

- later option

It should not be presented as part of CLARK's core proof case for the current investor narrative.

## Why

- The repo supports Dunkirk best as a possible logistics and warehousing option, not as a settled operating node.
- The Hamilton-Buffalo proof sequence does not require Dunkirk to be real on day one.
- Treating Dunkirk as active footprint creates avoidable scope and credibility risk.

## What CLARK Can Say

- Dunkirk remains a later logistics option worth validating if corridor volume and warehousing needs justify it.

## What CLARK Should Not Say

- Dunkirk is part of the active launch footprint.
- Dunkirk is already a committed logistics partner path.

## Trigger For Revisit

Revisit only if:

- corridor volume starts to support warehousing value
- Buffalo-side logistics constraints become material
- a credible property or public-sector path emerges with clear economics
