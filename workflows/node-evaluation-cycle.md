# Node Evaluation Cycle

## Objective

Evaluate a candidate node, corridor asset, or acquisition target in a repeatable way.

## Sequence

1. Create a profile using `templates/node-profile-template.md`
2. Validate location-specific claims in `docs/research.md`
3. Evaluate fit with current corridor and expansion strategy
4. Assess ownership, governance, and fee implications
5. Decide: core now, later option, or out of scope
6. Update `docs/decision-log.md` and relevant planning docs
